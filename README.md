# lol-rust-tools

Scripts to read and convert League file formats, but rewritten in Rust :)) 

Supported formats:

- inibin and troybin
- preload (future)
- luaobj and luabin (future)
- releasemanifest (future)

Supported converters:

- inibin to ini (future)
- luaobj to lua (future)
- troybin to troy
